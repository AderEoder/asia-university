# 我的個人部落格

這是一個使用 Flask 開發的簡易個人網站，用來介紹我自己與我們的撞球社。

## 🌐 線上預覽  
👉 https://yosoyleo.pythonanywhere.com/

## 🔧 使用技術  
- Python  
- Flask  
- HTML/CSS

## 📌 功能  
- 首頁介紹  
- 撞球社地址顯示  
- 使用者上傳圖片與文字資料  

## 📁 部署平台  
- PythonAnywhere
